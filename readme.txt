A Facebook page widget plugin with lot's of options

== Description ==

Numb Facebook Page Plugin is a very light plugin with lots of options. This is a widget that brings a facebook page details in WordPress website. There are lots of options that can change easily as per as needed.

Features:

*   Add Any Widget Area
*   Adapt Container Width Option
*   Show or Hide Timeline
*   Hide Cover Photo Options
*   Show or Hide Friends Faces
*   Show or Hide Call to Action Button (if available)
*   Set Width and Height Option ( Work only when Adapt container option is false )
*   Use Small Header Option

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Go to the appearance->widgets and select the "Numb Facebook Page Plugin" to add your desired widget area.
